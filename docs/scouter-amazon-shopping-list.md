## Notas Importantes

1. **Envío a Argentina:** El kit CanaKit suele enviar a AR. Los componentes sueltos verificar "International Shipping" en cada producto.
2. **Displays:** Se compraron 3 variantes a propósito para testear cuál funciona mejor con la óptica del beam splitter.
3. **ESP32-S3:** El pack de 3+1 cubre Gauntlet + Bridge + spare para prototipos.
4. **Baterías 18650:** Las Samsung 30Q son flat-top. Si necesitás button-top para ciertos holders, verificar.
5. **TP4056 + MT3608:** Los packs son grandes (3-10 unidades), cubren los 3 dispositivos y sobran para prototipos.
6. **Prioridad de compra:** RPi Zero 2W + displays + lente primero (para validar óptica Phase 0).
